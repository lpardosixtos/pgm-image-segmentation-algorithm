Para compilar:
g++ -std=c++11 main.cpp -o main

Para ejecutar:
Solo ejecutar, el programa  lee automaticamente los 5 archivos provistos en orden, solo lee 5 archivos.

Si se quiere usar otro archivo de prueba se debe modificar el nombre a Img0#.pmg con # un número entre 1 y 5.
El archivo pgm debe iniciar con las siguientes tres lineas.

P2
cols rows
255

Nota: Los archivos pgm fueron hechos en windows, cuando lo intentamos utilizar en ubunto era necesario renombrar aunque
el archivo quedara con el mismo nombre.
